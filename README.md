# Website for Project: Controllable Stereo Video Conversion with Guided Latent Decoding

This repository contains the source code for the project website of the paper:
**"Controllable Stereo Video Conversion with Guided Latent Decoding"**

## Website Template Usage

You are free to borrow the source code of this website for your own project page! We just ask that you link back to this repository or the original website in your footer.

The website features:
- **Interactive Anaglyph 3D Effects**: Hover effects on text and buttons.
- **Video Carousel**: A smooth, single-column video comparison gallery.
- **Comparison Slider**: Interactive split-screen video player for method comparison.
- **Showcase Video**: A hover-triggered input-output visualization switch.
